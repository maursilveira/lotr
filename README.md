# Lord of the Rings: Fellowship of the Ring

A responsive website about the movie

## Overview

This is a project for final integrated assignment during second term of Interactive Media Design program at Fanshawe College, London/ON, Canada. The goal of the assignment was designing and developing a responsive website about the movie, including two videos (a 15-second teaser and a 30-second trailer) also produced by students.
